Metodologi & Dataset
MODEL 1 · UMUR SIMPAN
Random Forest Regressor
"Beras pada 30°C dan RH 75% dapat bertahan 142 hari." Prediksi umur simpan berdasarkan komoditas + kondisi gudang. R² ≈ 0.988.
MODEL 2 · KEAMANAN
Random Forest Classifier
"Jagung: BERISIKO (78% confidence)." Klasifikasi AMAN / BERISIKO / TIDAK_AMAN dengan probabilitas per kelas. Accuracy ≈ 92.9%.
MODEL 3 · ANOMALI GAS
Isolation Forest
"Pola gas tidak normal." Dilatih pada kondisi normal. Sensitif terhadap lonjakan mendadak: kebakaran, LPG, fumigan.
Dataset: 3.000 sampel synthetic physics-based dari Arrhenius Q10=2 + ERH Labuza (1984) + threshold SNI / Codex Alimentarius. Valid sejak hari pertama. Noise sensor DHT11/MQ-2 disimulasikan (±0.5°C, ±1%RH, ±10ppm).
